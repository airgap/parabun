```ts
import simd from "parabun:simd";

const xs = new Float32Array(1_000_000);
const total = simd.sum(xs);
const scaled = simd.map(xs, x => x * 2 + 1);    // affine — fuses to one pass
```

## API

### Reductions

```ts
simd.sum(typed): number;
simd.min(typed): number;
simd.max(typed): number;
simd.mean(typed): number;
simd.dot(a, b): number;
simd.norm(typed): number;          // L2
```

### Element-wise

```ts
simd.map(typed, fn): typeof typed;          // fn must be pure + affine
simd.filter(typed, predicate): typeof typed;
simd.zip(a, b, fn): typeof a;               // pairwise
simd.scale(typed, k): typeof typed;         // a * k
simd.affine(typed, k, b): typeof typed;     // a * k + b
```

When `fn` is detected as affine (constant-folded coefficients) the call collapses to one SIMD pass. Otherwise the implementation falls back to a per-element loop in SIMD-vectorised form.

### Vector ops

```ts
simd.add(a, b): typeof a;
simd.sub(a, b);
simd.mul(a, b);
simd.div(a, b);
```

### Conversion / casting

```ts
simd.castF32ToF64(f32): Float64Array;
simd.castF64ToF32(f64): Float32Array;
simd.castI32ToF32(i32): Float32Array;
```

## Backends

| Platform | Vector width | Backend |
|---|---|---|
| x86_64 with AVX2 | 256-bit | Highway dispatch |
| x86_64 with AVX-512 | 512-bit (when available) | Highway |
| ARM64 NEON | 128-bit | Highway |
| Apple Silicon | 128-bit NEON + AMX where appropriate | Highway |
| Fallback | scalar loop | Highway portable |

Throughput is typically 4-8× scalar Float32 for affine kernels.

## Pipeline fusion

When used inside a `|>` chain, `simd.map` / `simd.affine` chains collapse via `parabun:pipeline`:

```ts
const result = data
  |> _.map(x => x * 2)
  |> _.map(x => x + 1)
  |> _.filter(x => x > 100);
// → one fused SIMD pass
```

See [`parabun:pipeline`](/parabun/runtime/pipeline) for fusion details.
